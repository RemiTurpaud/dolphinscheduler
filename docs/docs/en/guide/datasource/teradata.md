# Teradata Vantage

## Datasource Parameters

|       **Datasource**       |                       **Description**                       |
|----------------------------|-------------------------------------------------------------|
| Datasource                 | Select TERADATA.                                            |
| Datasource name            | Enter the name of the DataSource.                           |
| Description                | Enter a description of the DataSource.                      |
| IP/Host Name               | Enter the Teradata host name.                               |
| Port                       | Enter the Teradata service port.                            |
| Username                   | Set the username for Teradata connection.                   |
| Password                   | Set the password for Teradata connection.                   |
| Database name              | Enter the database name of the Teradata connection.         |
| Jdbc connection parameters | Parameter settings for Teradata connection, in JSON format. |

## Native Supported

Yes, could use this datasource by default.
